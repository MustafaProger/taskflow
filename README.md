# New Knowledge

Это личная база заметок по проекту.
Здесь собраны вопросы и ответы, которые появляются по ходу изучения React, TypeScript, Tailwind, CSS и инструментов проекта.

Основные материалы лежат в папке [`new-knowledge`](./new-knowledge).

## Как устроена структура

- в папке `new-knowledge` каждая тема вынесена в отдельный файл
- внутри каждого файла сначала идет список вопросов
- каждый вопрос сверху ведет к своему ответу ниже
- структуру можно спокойно расширять новыми вопросами по тем же темам

## Темы

- [Tooling](./new-knowledge/Tooling.md) - вопросы про инструменты проекта и сборку
- [CSS](./new-knowledge/CSS.md) - вопросы про CSS
- [Tailwind](./new-knowledge/Tailwind.md) - вопросы про Tailwind
- [TypeScript](./new-knowledge/TypeScript.md) - вопросы про TypeScript
- [React](./new-knowledge/React.md) - вопросы про React, структуру компонентов и роли вроде `ui`, `layout`, `feature`
